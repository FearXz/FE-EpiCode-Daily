Eventi js

un evento è un qualcosa che accade all'interno della pagina
un utente effettua delle azioni che bisogna intercettare

onload

quando carica il body , praticamente sempre quando apri la pagina

onmouseover

al passaggio del mouse sull'area scelta scatta evento

onmouseout

quando il mouse esce dall'area scelta scatta evento

onclick

quando clicco scateno l'evento

Altri

onkeypress (pressione tasto) - onchange (quando un elemento subisce un cambiamento)
- onsubmit (invio diun form) - onfocus|onblur (per evidenziare che un elemento è attivo)
(doubleclick - drag and drop - e tanti altri)

Eventi ed Handler

dobbiamo intercettari gli eventi , per farlo si usano gli handler o listener , a questi assoceremo una funzione di callback
cioè che si richiama quando ha intercettato un evento.

  • via codice con addeventlistener()
  • via html con speciali tag
  • o sempre in javascript associando funzioni al DOM

addEventListener()

il modi più comune e migliore per associare evento al suo handler

Evento

gestisce il tipo di evento che vogliamo gestire (click onload ...)

callback

chiamata o invocazione della funzione che dobbiao attivare

useCapture

è opzionale , usato raramente ed è false di default , viene usato per forzare le priorità di un certo evento
se associamo più handler allo stesso evento metto true al primo da usare

Parametro Event

è un oggetto che contiene informazioni sull' evento

i vantaggi di addEventListener

  • funziona su tutti gli elementi del DOM
  • possiamo gestire più funzioni per lo stesso evento
  • possiamo intercettare più eventi sullo stesso elemento
  • possiamo rimuovere l'associazione tra evento ed handler usando removeEventListener()

è sconsigliato utilizzare le funzioni eventi direttamente nell' html - html css e js devono stare separati
