LOGICA CONDIZIONALE

qualunque script di istruzione eseguite dall'engine javascript
script: insieme di comandi che stabiliscono il comportamento del sistema o algoritmo.

nel codice condizionale eseguiamo algoritmi o blocchi di codice {} se si verifica una condizione

IF - IF ELSE - ELSE

IF

ci fa eseguire un blocco di codice se la condizione è verificata.

IF ELSE

ci fa eseguire un blocco di codice se la condizione è verificata, se la condizione non è verificata , allora verifica un altra condizione, se quest'ultima è verificata esegue un secondo blocco di codice.

ELSE

ci fa eseguire un blocco di codice se nessuna delle altre condizioni è verificata.
